ColumnCombine

Combines to columns into a single column by keeping the values of the first column, when those values are not missing, and taking the values of the second column only when the first columns has a missing value.

For other forms of combining columns, such as addition, subtraction, etc. use ColumnOp.

Options

columns: ColumnName(s) to be combined, either a single column or a list of columns.
keepNonNull: Flag to keep non-null values while combining columns. Must be used

Examples

Example 1

#> ColumnCombine First Person Rating Second Person Rating --keepNonNull
AFLEFT 
modifiedPizzeriasDf['First Person Rating'] = modifiedPizzeriasDf['First Person Rating'].combine_first(modifiedPizzeriasDf['Second Person Rating'])
