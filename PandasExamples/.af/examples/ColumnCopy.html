ColumnCopy

Copies a column within a dataframe to an additional column with a new name. This is beneficial to use when you want to perform operations on the data in a column but do not want to modify or overwrite the original data.

Options

columns: Specifies columns to copy
to: Specifies the new column name

Examples

Example 1

#> ColumnCopy Location
AFLEFT 
pizzeriasDf['LocationCopy'] = pizzeriasDf['Location'] AFRIGHT

Example 2

#> ColumnCopy Specialty Pizza --to Old Specialty Pizza
AFLEFT 
pizzeriasDf['Old Specialty Pizza'] = pizzeriasDf['Specialty Pizza'] AFRIGHT

Example 3

#> ColumnCopy Rating Website --to PreviousRating PreviousWebsite
AFLEFT 
pizzeriasDf['PreviousRating'] = pizzeriasDf['Rating']
pizzeriasDf['PreviousWebsite'] = pizzeriasDf['Website'] AFRIGHT
