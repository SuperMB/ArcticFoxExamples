ColumnFor

Creates a very basic for loop architecture for looping over the columns, it leaves the contents of the for loop for the user to fill out, but should be used only to create a for loop structure. The should not be used for grouping procedures

Options

index: Specifies the index to use

Examples

Example 1

#> ColumnFor
AFLEFT 
for column in bankTransactionsDf.columns: AFRIGHT
