Visualize

Prints the text value of a dataframe from the head function into either the file, if a .py file, or into a notebook cell. Not used for graphing data, just used to print the text value of head() for a dataframe.

Options

count: Specifies a count plot or visualization

Examples

Example 1

#> Visualize
AFLEFT print(appleStockDf.head()) #)1 AFRIGHT

Example 2

#> Visualize --count 10
AFLEFT print(appleStockDf.head(n=10)) #)2 AFRIGHT
