ColumnUnique


Gets a list of the unique values that appear in a column. This reveals what categories or values are in a column, if there are values that you did not expect to appear, and if there are values that are improperly formatted. While it can be used with numeric, categorical, date, etc. datatypes, it is less useful for columns with a large number of different values.

Options

columns: Specifies columns to round

Examples

Example 1

#> ColumnUnique CustomerAge --print
AFLEFT 
bankTransactionsDfUnique = bankTransactionsDf['CustomerAge'].unique()
print(bankTransactionsDfUnique) #)2 AFRIGHT

Example 2

#> ColumnUnique TransactionType Location
AFLEFT 
bankTransactionsDfUnique_1_TransactionType = bankTransactionsDf['TransactionType'].unique()
bankTransactionsDfUnique_1_Location = bankTransactionsDf['Location'].unique() AFRIGHT

Example 3

#> ColumnUnique --print
AFLEFT 
bankTransactionsDfUnique_1 = bankTransactionsDf.nunique()
print(bankTransactionsDfUnique_1) #)7 AFRIGHT
