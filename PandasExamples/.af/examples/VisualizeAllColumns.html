VisualizeAllColumns


By default, VisualizeAllColumns is used with the Visualize kit. This kit instructs the Visualize kit to display all of the columns as oppose to the default the replaces columns with ... if the output is too wide. Code is written to the top of the code file to implement the desired effect.

Examples

Example 1

#> VisualizeAllColumns
AFLEFT
